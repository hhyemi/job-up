# job-up (직업을 잡아)
취업준비생을 위한 달력, TODO-LIST를 이용한 일정관리 웹사이트 :memo:  
일정관리뿐만 아니라 간단히 기록할 수 있는 메모기능 , 스탑워치를 통해 주간, 일간 공부기록이 가능하다.  
커뮤니티기능을 통해 사람들과 취업정보공유가 가능하다.
